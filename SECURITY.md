# Security Policy

## Supported versions
This repository tracks a rolling main branch with generated artifacts.

## Reporting vulnerabilities
Please open a private security advisory in GitHub Security tab, or contact maintainers directly if available.

## Scope notes
- This project publishes network lists and export formats.
- No secrets should be committed into the repository.
- CI/release workflows are deterministic-oriented; unexpected non-deterministic output should be reported as a supply-chain concern.
